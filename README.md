

# Self Attendance Tracker

A simple personal application to record daily attendance and track consistency over time.  
This project focuses on accountability through data — not motivation, not reminders, just records.

##  Purpose
The goal of this tool is to track whether I show up daily and measure consistency objectively.  
It eliminates guesswork by storing attendance records and calculating real statistics.

##  Features
- Record daily attendance (Present / Absent)
- Automatically save records
- View attendance history
- Calculate attendance percentage
- Lightweight and fast
